# 초안 형식·톤 가이드 (4종)

모든 초안은 `# 제목` H1으로 시작(노션 페이지 제목). 근거는 `docs/marketing-plan.md`.
3대 메시지 축 중 하나 이상에 귀속: **편의 / AI 차별점 / 페르소나(일반·스몰·셀프)**.

---

## 1. 네이버 블로그 글 (`blog-*.md`)

**목표:** 검색 유입(SEO). **정보성 70% : 앱 소개 30%.** 노골적 광고 톤 금지.

**구조:**
- `#` 제목 — 롱테일 키워드를 앞단에 (예: "2026 결혼 준비 순서 체크리스트 (예비부부 필독)")
- 도입: 공감 한 문단 (검색자의 고민)
- `##` 소제목 4~6개로 정보 전개 — 시기별/항목별 표·리스트 활용
- 본문 1,500자+ / 이미지 자리표시 `> [이미지: 앱 체크리스트 화면]` 8개+
- 마지막 `##` 에 자연스러운 앱 연결 1문단 + CTA: `> 👉 이 체크리스트, Dewy 앱에서 자동 관리`
- 하단: 해시태그/관련 키워드 줄

**톤:** 경험 공유형("저는 이렇게 했어요"), 과장·단정 금지.

---

## 2. 네이버 카페 글 (`cafe-*.md`)

**목표:** 신뢰 적립 → 자연 유입. **홍보 최소화, 도움 우선.** 강퇴/차단 리스크 인지.

**유형 2가지 (요청에 맞게):**
- **질문 답변형:** 다른 예비부부 고민에 실질적 답 + 끝에 "저는 Dewy 앱으로 ~했는데 편했어요" 한 줄 맥락 삽입(선택)
- **후기형:** 내 준비 경험 공유 중 앱 사용을 자연스럽게 1~2회 언급. 별점·과한 칭찬 금지.

**구조:** 짧은 제목 / 구어체 본문 / 표·리스트 가볍게 / 끝에 무료 가치 미끼(예: "체크리스트 PDF 필요하면 댓글 주세요").

**금지:** 자작 후기, 다중계정, 링크 도배. 카페 규정 우선.

---

## 3. 숏폼 스크립트 (`shortform-*.md`) — 클립/릴스/쇼츠 공용

**목표:** 비주얼 임팩트로 인지도. 7~15초, 세로 9:16, **첫 1초 후킹 + 브랜드명 노출.**

**구조(각 스크립트):**
- `#` 제목 (= 영상 콘셉트)
- `## 후킹 (0~1초)` — 첫 자막/멘트 1줄
- `## 장면 구성` — 컷별 표: `| 초 | 화면 | 자막/내레이션 |`
- `## CTA` — "프로필 링크 / 지금 무료로 시작"
- `## 해시태그` — #예비신부 #결혼준비 #스몰웨딩 #셀프웨딩 #청첩장 #스드메
- `## 음원/편집 메모` (선택)

**킬러 소재:** 드레스 투어 before→after, AI 메이크업 변신, 청첩장 0원→완성 타임랩스, 예산 평균 비교, 페르소나 전환.

## 4. 광고 카피 (`ad-*.md`) — 구글/메타

**목표:** 설치/전환. 15초 이내 영상 카피 + 정적 캐러셀 카피.

**구조:**
- `#` 캠페인/소재명
- `## 구글 (유튜브/UAC)` — 헤드라인(30자×3), 설명(90자×2), 영상 5초 후킹 스크립트
- `## 메타 (인스타·페이스북)` — 1차 텍스트(125자 내), 헤드라인, 캡션, 캐러셀 3~5장 각 슬라이드 카피
- `## 타겟 메모` — 한국 25~40, 라이프이벤트/관심사(결혼·약혼·웨딩), Lookalike
- `## A/B 변형` — 같은 메시지 2~3안

**원칙:** 첫 줄/첫 5초에 핵심 베네핏 + Dewy 노출. 과장·허위 효능 금지. CTA "지금 무료로 시작".
