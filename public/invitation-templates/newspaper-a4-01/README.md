# Newspaper A4 4P template

Admin registration:

1. Open `/admin/invitation-templates` and choose `새 템플릿`.
2. Set slug to `newspaper-a4-01`, format to `paper`, and upload `thumb.png`.
3. Use `JSON 불러오기` with `layout.json`.
4. The four page background upload fields appear automatically. Upload `p1-cover.png`, `p2-love-story.png`, `p3-itinerary.png`, and `p4-interviews.png`.
5. Save the template.

The canvas dimensions use the supplied wireframe resolution (`1055 x 1490`).
Every page exports at A4 finished size (`210 x 297mm`).
