# Dewy

Dewy is an AI wedding planning application built with React, Vite, TypeScript, Supabase, and Capacitor for Android.

## Local Development

Install dependencies:

```sh
npm install
```

Run the Vite development server:

```sh
npm run dev:vite
```

Build the web app:

```sh
npm run build:vite
```

Run tests:

```sh
npm run test
```

Build and sync Capacitor assets:

```sh
npm run cap:build
```

## Tech Stack

- React 18
- Vite
- TypeScript
- Tailwind CSS
- Supabase
- Capacitor Android
