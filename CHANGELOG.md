# 변경 이력 (Changelog)

앱 버전(`android/app/build.gradle` 의 `versionName`/`versionCode`) 기준.

## 2.2.0 (versionCode 7) — 2026-06-15

견적 → 연결 → 예약 → 플래닝까지 전 과정을 하나로 잇고, **내 업체 보드·비교**로 결정을 정리하는 릴리스.

**견적·연결 (핵심)**
- 견적 요청 후 **진행 상태 트래커**로 어디까지 왔는지 한눈에 (소비자 가시화)
- 견적 수락 시 **양방향 연락 연결** + **인앱 실시간 메시지**(소비자↔업체 앱 내 대화)
- **견적 비교 UX** 강화 — 응답 카드에 평점·제휴·지역 등 신뢰 신호 표시
- 견적 요청에 **참고 사진 첨부**(최대 3장)
- **매칭 랭킹 고도화** + 업체 ROI 퍼널(예약 전환까지 추적)
- 미응답 **SLA 독려 알림** + 알림 **미읽음 배지**

**예약·플래닝 연동**
- 예약 완료 → **내 예산에 자동 반영**
- 예약 완료 → **일정(체크리스트)에도 반영**
- 내 견적에 **예약완료 표시** + 미응답 재요청 안내

**내 업체 보드 · 비교 (신규)**
- **내 업체 보드** — 베뉴·스튜디오·드레스·스냅·청첩장 등 카테고리별 미정/견적중/예약완료 상태와 선택 업체를 한 보드에서 정리(받은 견적 자동 반영)
- **업체 비교** — 찜한 후보를 나란히 비교해 결정
- 홈 상단 **빠른접근**(보드·견적·비교·일정·예산·찜) + 마이페이지 진입점으로 기능 발견성 개선
- 첫 방문 보드 온보딩 안내 · 삭제 확인 등 UX 안전장치

**안정화·내부**
- 견적/업체/관리자 도메인 **타입 안전화**(as-any 제거)로 런타임 회귀 차단
- 전반 코드 감사 · 다수 버그 수정

### Play Store "이번 업데이트" 문구
```
- 여러 업체 견적을 한 곳에서 비교하고, 수락하면 앱에서 바로 채팅·연락 연결
- 견적 진행 상태를 한눈에 + 미응답 알림으로 놓치지 않기
- 예약하면 예산·일정에 자동 반영
- '내 업체 보드'로 베뉴·드레스·스냅까지 준비 상태를 한 보드에서 정리
- 업체 비교·홈 빠른접근 추가 및 안정성 개선
```

## 2.1.0 (versionCode 6) — 2026-06-15

대규모 기능·정합성 릴리스. (versionCode 6 은 미게시 상태로 2.0.3 준비분을 함께 포함.)

**연결·업체**
- 소비자↔제휴업체 **견적 요청/매칭**(조건 남기면 여러 업체가 견적) · 업체 응답 신뢰 신호(빠른응답·응답률) · 수요 가시화 입점 유도
- 미입점 업체 **'문의하기'** 실연결(온라인 채널→전화→안내) · 입점 업체 문의 채널 선택(앱채팅/URL/전화)
- **업체 정보 승인 버그 수정**(승인이 안 되던 P0) · 후기 답변 · 고객 후기 대시보드

**기능**
- 청첩장 만들기 **3-모드**(AI 자동생성·템플릿·자유) · 캔버스 편집 개선
- AI **결과물 통합 페이지**(헤어·드레스·메이크업·사진보정·컨설팅 탭) · 태그로 업체 검색
- 홈 진입 팝업 개선(크게·하루/일주일 보지않기) + 운영자 이벤트/팝업 관리

**안정화·정합성**
- 출석체크 완료 표시 버그 수정 · 결제 중복발사 방지 · 데이터 신선도 지표 정확화
- 보안 강화(권한·어뷰징 방지) · 다수 버그 수정

### Play Store "이번 업데이트" 문구
```
- 여러 업체에서 한 번에 견적 받기 (견적 매칭)
- 청첩장 만들기 새 단장 (AI 자동생성·템플릿·자유 편집)
- AI 결과물(헤어·드레스·메이크업·보정·컨설팅) 한 곳에서 모아보기
- 업체 문의·연결 개선 및 응답 신뢰 표시
- 출석·결제·데이터 안정성 개선 및 다수 버그 수정
```

## 2.0.2 (versionCode 5) — 2026-06-13

결제 안정화 · FE/BE 정합성 버그 수정 · 백엔드 정리 릴리스.

### 사용자 영향 (Play Store "이번 업데이트" 후보 문구)
- **카카오페이 결제 안정화** — 스토어 주문 결제를 카카오페이로 일원화(토스페이먼츠 제거). 서버에서 금액을 다시 계산·검증하고 불일치 시 자동 취소하도록 보강.
- **포인트 잔액 표시 정확도 개선** — 게임 화면·마이페이지·AI 챗봇의 포인트가 실제 사용 가능 잔액과 어긋나던 문제 수정(사용분 미반영 → 정상 반영).
- **AI 챗봇 추천 개선** — "예복" 검색이 결과 0건이던 문제, "내 지역 웨딩홀 수" 안내가 안 뜨던 문제 수정.
- 여러 버그 수정 및 표기 일관성 개선.

### 개발/백엔드 (사용자 비노출)
- 스토어 결제 경로 카카오페이 신설(`kakao-pay-order-ready`/`approve`) + 토스 함수·SDK·env 제거.
- FE↔BE 값 정합성 버그 5건 수정: 포인트 `total_points`→`balance`, 어드민 데이터 신선도 카테고리(`suit`→`tailor_shop`, 누락 `invitation_venue`/`appliance` 추가), 챗봇/AI플래너 `suit`→`tailor_shop` 변환, 죽은 `venues` 참조→`places` 복구.
- 가격 표기 단일화(`formatWon`, 9곳 복붙 제거).
- 레거시 빈 테이블 6종 드롭(`appliances/hanbok/honeymoon/honeymoon_gifts/studios/suits`).
- Supabase 타입 재생성(실 DB 113테이블 동기화 — 스테일 제거, 누락 29개 보강).
- FE/BE 값 정합성 CI 가드 신설(`scripts/check-integrity.mjs`) — 회귀 자동 차단.
- 전 페이지 고도화 코드리뷰 문서(`docs/260613_codereview_2.md`).
