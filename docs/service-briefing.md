# Dewy(듀이) 서비스 브리핑

> 다른 AI(제미나이 등)에게 이 서비스 컨텍스트를 한 번에 전달할 때 사용하는 통합 브리핑.
> 그대로 복사·붙여넣기 하시면 됩니다.
>
> 최종 검증: 2026-05-20 (코드 vs 문서 감사 반영)

---

## 한 줄 소개

**Dewy(듀이)는 결혼을 준비하는 한국 커플을 위한 AI 통합 플래너 앱입니다.**
복잡한 결혼 준비 과정 — 정보 조사, 예산 관리, 일정 추적, 업체 비교, 드레스 시연,
커뮤니티 소통, 결제 — 을 한 곳에서 해결합니다.

---

## 타겟 사용자

- **주 타겟**: 25~45세 결혼 예정자 (예비 신랑·신부)
- **부 타겟**: 신혼부부, 결혼 준비 시기 부모
- **지역**: 한국 (한국 웨딩 시장 도메인 특화 — 스드메, 예단, 보증인원 등)

---

## 핵심 차별점

1. **AI 우선 설계** — Google Gemini 2.5 Flash 기반 한국 웨딩 특화 챗봇이 24시간 상담사 역할
2. **사진 기반 AI 드레스 시연** — 발품 팔기 전 드레스를 사용자 사진에 입혀보기
   (OpenAI gpt-image-2 이미지 편집 모델 사용)
3. **커플 동시 사용** — 두 사람이 같은 데이터(예산·일정·즐겨찾기)를 함께 보고 수정
4. **통합 플랫폼** — 정보 → 플래닝 → 커뮤니티 → 쇼핑이 한 앱에서

## 주요 기능

### 🤖 AI 플래너
- Google **Gemini 2.5 Flash** 기반 한국 웨딩 특화 챗봇
- 무료: 일 5회, Premium: 무제한
- 예산·일정·지역·선호도 기반 맞춤 답변
- **장기 메모리** — 사용자의 결혼 정보(예식일·지역·예산 등)를 학습해 후속 대화에 반영
- **AI 오류 처리** — Gemini 장애 시 명확한 서비스 오류 응답 반환
- PDF 견적서·분석 리포트 자동 생성 (Premium)

### 📸 AI 스튜디오
- **방구석 드레스 투어** — 사용자 사진에 다양한 드레스 입혀보기 (OpenAI gpt-image-2, 5 하트)
- **착붙 메이크업 찾기** — 메이크업 시연 (5 하트)
- **모바일·종이 청첩장 자동 생성** (2 하트)
- **웨딩 촬영 시안** (3 하트)
- **식전 영상 외주** (정액제)

### 🏛️ 정보·조회
- 카테고리: 웨딩홀 / 스튜디오 / 드레스 / 메이크업 / 한복 / 정장 / 신혼가전 / 예물·예단 / 식대 / 신혼여행 (9개)
- 업체 갤러리, 가격, 위치, 평점, SNS 연동
- 웨딩 인플루언서 정보 (Instagram, YouTube, 블로그)

### 📊 플래닝 도구
- 예산 관리 (도넛 차트, 카테고리별 평균치 비교)
- 일정 관리 (D-day 자동 역산, 골든타임 알림)
- 체크리스트 (시기별 자동 정렬)
- 하객 명단 관리 (RSVP, 동반자, 신랑/신부 구분)

### 💑 커플 기능
- 초대 코드 기반 파트너 계정 연동
- 커플 다이어리 (결혼 준비 기록 공유)
- 커플 투표 (식장·콘셉트·드레스 등 의사결정)
- 즐겨찾기 동기화

### 💬 커뮤니티
- 카테고리별 게시판 (웨딩홀·스드메·허니문·혼수·자유)
- 스타일별 필터 (일반·스몰웨딩·셀프웨딩)
- 게시글·댓글·이미지 업로드, 북마크, 좋아요

### 🎮 캐주얼 게임 (보너스 콘텐츠)
- **MergeGame** — Matter.js + Konva 기반 머지 퍼즐. 결혼 준비 스트레스 해소 + 리텐션 장치

### 🛠️ 어드민 대시보드 (사내 운영용)
- 사용자·드레스·메이크업 샘플 관리
- 서비스 대기열 관리
- B2B 사업자 검수

### 💎 수익 모델
- **하트 충전** (인앱결제, Toss Payments)
  - 첫 구매(스타터): 1,900원 = 10 하트 (단가 190원)
  - 추가 패키지: 4,900원 / 9,900원 / 19,900원 / 49,900원 등 (`src/lib/heartPackages.ts` 참조)
- **Premium 구독** (Kakao Pay)
  - **월간 4,900원** / **연간 39,000원**
  - AI Planner 무제한, PDF 리포트 무제한
- **포인트** — 가입 시 신규 가입 보너스(`signup_bonus`) 자동 적립
- **B2B (사업자용)**
  - 업체 등록·갤러리 관리·문의 관리·통계 대시보드

## 기술 스택

- **프론트엔드**: React 18 + Vite 6 + TypeScript 5
- **UI**: Tailwind CSS + shadcn-ui (Radix UI 기반)
- **모바일 패키징**: Capacitor 6 (Android 네이티브 wrapping)
- **백엔드**: Supabase (PostgreSQL + Auth + Storage + Edge Functions)
- **AI (챗봇)**: Google Gemini 2.5 Flash
- **AI (이미지 편집)**: OpenAI gpt-image-2
- **결제**: Toss Payments (하트 충전), Kakao Pay (구독)
- **호스팅**: Vercel (웹), Google Play (Android 앱)
- **인증**: Google OAuth, Kakao OAuth (네이티브 PKCE flow)
- **푸시 인프라**: FCM 연동 Edge Function 작성 완료, 1차 출시에선 미활성화

## 데이터 처리 원칙

- 모든 데이터 암호화 저장 (Supabase 기본 + TLS)
- AI 업로드 사진은 **처리 후 30일 자동 삭제** — `cleanup-ai-uploads` Edge Function 이
  pg_cron 으로 매일 1회 실행되어 `dress-uploads`·`dress-results` 버킷에서
  30일 초과 파일을 Storage API 로 안전하게 제거 (`docs/ai-uploads-retention.md`)
- 결제 정보는 앱에 보관하지 않음 (Toss / Kakao Pay 직접 처리)
- 회원 탈퇴 시 즉시 데이터 파기 (법정 보관 정보 제외)
- 14세 미만 가입 차단 (정책상 — 가입 폼 검증 구현은 출시 전 확인 항목)
