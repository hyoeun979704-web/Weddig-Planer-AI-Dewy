# Dewy(듀이) 서비스 브리핑

> 다른 AI(제미나이 등)에게 이 서비스 컨텍스트를 한 번에 전달할 때 사용하는 통합 브리핑.
> 그대로 복사·붙여넣기 하시면 됩니다.

---

## 한 줄 소개

**Dewy(듀이)는 결혼을 준비하는 한국 커플을 위한 AI 통합 플래너 앱입니다.**
복잡한 결혼 준비 과정 — 정보 조사, 예산 관리, 일정 추적, 업체 비교, 드레스 시연,
커뮤니티 소통, 결제 — 을 한 곳에서 해결합니다.

---

## 타겟 사용자

- **주 타겟**: 25~45세 결혼 예정자 (예비 신랑·신부)
- **부 타겟**: 신혼부부, 결혼 준비 시기 부모
- **지역**: 한국 (한국 웨딩 시장 도메인 특화 — 스드메, 예단, 보증인원 등)

---

## 핵심 차별점

1. **AI 우선 설계** — Google Gemini 기반 한국 웨딩 특화 챗봇이 24시간 상담사 역할
2. **사진 기반 AI 시연** — 발품 팔기 전 드레스·메이크업을 사용자 사진에 입혀보기
   (Google Nano Banana Pro 2 이미지 생성 모델 사용)
3. **커플 동시 사용** — 두 사람이 같은 데이터(예산·일정·즐겨찾기)를 함께 보고 수정
4. **통합 플랫폼** — 정보 → 플래닝 → 커뮤니티 → 쇼핑이 한 앱에서

---

## 주요 기능

### 🤖 AI 플래너
- Google Gemini 기반 한국 웨딩 특화 챗봇
- 무료: 일 5회, Premium: 무제한
- 예산·일정·지역·선호도 기반 맞춤 답변
- PDF 견적서·분석 리포트 자동 생성 (Premium)

### 📸 AI 스튜디오 (6가지 디지털 서비스)
- 방구석 드레스 투어 (사용자 사진에 드레스 입히기, 5 하트)
- 착붙 메이크업 찾기 (5 하트)
- 모바일/종이 청첩장 자동 생성 (2 하트)
- 웨딩 촬영 시안 (3 하트)
- 식전 영상 외주 (정액제)

### 🏛️ 정보·조회
- 8대 카테고리: 웨딩홀 / 스튜디오 / 드레스 / 한복 / 정장 / 신혼가전 / 예물 / 신혼여행
- 업체 갤러리, 가격, 위치, 평점, SNS 연동
- 웨딩 인플루언서 정보 (Instagram, YouTube, 블로그)

### 📊 플래닝 도구
- 예산 관리 (도넛 차트, 6개 카테고리, 평균치 비교)
- 일정 관리 (D-day 자동 역산, 골든타임 알림)
- 체크리스트 (시기별 자동 정렬)
- 하객 명단 관리 (RSVP, 동반자, 신랑/신부 구분)

### 💑 커플 기능
- 초대 코드 기반 파트너 계정 연동
- 커플 다이어리 (결혼 준비 기록 공유)
- 커플 투표 (식장·콘셉트·드레스 등 의사결정)
- 즐겨찾기 동기화

### 💬 커뮤니티
- 카테고리별 게시판 (웨딩홀·스드메·허니문·혼수·자유)
- 스타일별 필터 (일반·스몰웨딩·셀프웨딩)
- 게시글·댓글·이미지 업로드, 북마크, 좋아요

### 💎 수익 모델
- **하트 충전** (인앱결제, Toss Payments)
  - 첫 구매 1,900원 = 10 하트
  - 가입 시 1,000P 적립
- **Premium 월 구독** (Kakao Pay)
  - AI Planner 무제한
  - PDF 리포트 무제한 생성
- **B2B (사업자용)**
  - 업체 등록·갤러리 관리·문의 관리·통계 대시보드

## 기술 스택

- **프론트엔드**: React + Vite + TypeScript
- **모바일 패키징**: Capacitor (네이티브 안드로이드 앱 wrapping)
- **백엔드**: Supabase (PostgreSQL + Auth + Storage + Edge Functions)
- **AI**: Google Gemini API (챗봇), Google Nano Banana Pro 2 (이미지 생성)
- **결제**: Toss Payments, PortOne, Kakao Pay
- **호스팅**: Vercel (웹), Google Play (Android 앱)
- **인증**: Google OAuth, Kakao OAuth (PKCE flow on mobile)
