## 1. 사업 개요

### 가. 사업 명칭 및 목적

본 사업의 명칭은 '듀이(dewy)'이며, **사용자별 초개인화(Hyper-personalization)를 핵심 가치로 하는 결혼 준비 큐레이션 플랫폼** 운영을 목적으로 한다.

브랜드명 '듀이(dewy)'는 'Duo(둘) + Easy(쉽게)'의 합성어로, "둘이서 쉽게 준비하는 웨딩"을 의미한다. 본 사업은 페르소나·진행 단계·지역·일정·특이사항(예: 임신·재혼 등) 등 다차원 데이터를 기반으로, 각 사용자에게 자신의 결혼 준비에 필요한 정보만을 선별 노출하여 의사결정 부담을 최소화한다.

결혼은 본 사업의 첫 적용 도메인이며, 향후 라이프스테이지 전반(신혼·영유아 등)으로 동일한 초개인화 엔진을 확장한다. 서비스 도메인은 dewy-wedding.com이다.

### 나. 주요 사업 영역

1. 초개인화 큐레이션 기반 결혼 준비 정보 제공
2. AI 챗봇 기반 웨딩 플래닝 상담 서비스
3. AI 이미지 생성을 활용한 드레스·메이크업 시뮬레이션
4. 청첩장 및 시안 자동 생성 서비스
5. 결혼식장·스튜디오·허니문 등 통합 정보 제공
6. 예산·일정·체크리스트 디지털 관리 도구
7. 사업자(B2B) 셀프 입점·관리 콘솔

### 다. 주요 특징

1. **초개인화 큐레이션 엔진**: 페르소나 기반 모드 선택 + 다차원 데이터로 사용자별 다른 정보 노출
2. **한국 웨딩 도메인 특화 AI**: 스드메·예단·예물·보증인원 등 도메인 지식 내장
3. **AI 시각화**: 사용자 사진 기반 드레스·메이크업 시뮬레이션
4. **통합 플랫폼**: 정보·플래닝·커뮤니티·커머스·B2B·AI 서비스를 단일 앱에서 제공
5. **자동화 기반 1인 운영**: AI Studio 6개 서비스 중 5개 자동 처리

### 라. 운영 모델 (요약)

대표자 1인 부업 형태로 출시한다. 대표자는 본업에서 개발·디자인·UI/UX·마케팅·촬영·영상 편집 직무를 수행하고 있으며, 동 역량을 사업 운영에 직접 활용한다. 외부 자금·외부 인력 투입 없이 자체 자본 및 자체 매출을 기반으로 점진적으로 성장하는 구조이다.

## 2. 시장 환경

### 가. 시장 규모

| 지표 | 추정 규모 | 트렌드 |
|---|---|---|
| 연간 혼인 건수 | 약 19~20만 건 | 감소세 (2014년 대비 약 40% 감소) |
| 평균 결혼식 비용 | 3,000~5,000만원 | 증가 |
| 1인당 디지털 웨딩 지출 | 30~80만원 | 증가 |
| 모바일 청첩장 시장 | 약 200~300억원 | 정체 (포화) |
| 디지털 웨딩 도구 시장 (TAM) | 약 1,500~2,000억원 | 성장 |
| AI·플래너 디지털 도구 (SAM) | 약 200~400억원 | 빠른 성장 |

**출처 및 산정 근거**
- 연간 혼인 건수: 통계청 「인구동향조사(혼인·이혼)」 기준 (2023년 약 19.4만 건)
- 평균 결혼식 비용: 결혼정보업체 듀오 「결혼비용 보고서」 등 공개 자료 종합
- TAM/SAM: 공신력 있는 단일 출처가 부재한 신생 영역으로, **혼인 건수 × 1인당 디지털 웨딩 지출 × 디지털 침투율**로 산정한 자체 추정치

### 나. 시장 트렌드

1. **초개인화 수요 증가**: 정보 과다 시대에 사용자는 "더 많은 정보"가 아닌 "나에게 맞는 정보"를 원함. Spotify, Netflix, TikTok, 당근마켓 등의 성공이 이를 입증.
2. **AI 도입 확산**: 예비 부부의 ChatGPT·Claude 등 일반 LLM 활용이 증가하고 있으며, AI 시각화 서비스에 대한 수요가 빠르게 형성.
3. **결혼 형태 다양화**: 스몰웨딩 수요 증가 (2015년 대비 약 3~4배), 셀프웨딩·노웨딩 트렌드 확산.
4. **인구 통계 변화**: 비혼·만혼 증가, 임신 결혼·재혼·국제결혼 등 비표준 케이스 비중 증가.
5. **모바일 우선 의사결정**: 결혼 준비 정보 검색 및 의사결정의 70% 이상이 모바일에서 발생.
6. **시각 자료 중심 의사결정**: 인스타그램·핀터레스트 등 이미지 중심 플랫폼이 의사결정에 큰 영향.
7. **합리적 소비 강화**: 비교·견적·후기 기반의 의사결정이 보편화.

### 라. 결혼식 직후 전환 메커니즘 — 사용자 선택형 안내

결혼식 D-Day 자동 인지 시점에 축하 메시지와 함께 사용자가 다음 단계를 직접 선택하는 안내 화면을 제공한다. 강제 CTA가 아닌 사용자 의도 기반 진입으로 피로도를 최소화한다.

```
[결혼식 축하 메시지]
    ↓
"다음 단계로 무엇을 도와드릴까요?"

  ⓐ 결혼 용품 정리하고 싶어요  →  중고거래 안내 (1년차: 당근 검색 자동 생성)
  ⓑ 신혼 추억 남기고 싶어요    →  라이프 스냅 모듈 (4-3 참조)
  ⓒ 혹시 곧 가족이 늘어나나요? →  자녀계획 수집 → 노리(가칭, 영유아 단계 전용 별 도메인) (조건부)
  ⓓ 결혼 후 정보만 받고 싶어요 →  기념일·이벤트 알림 모드
```

각 선택지 진입 시 포인트 보상으로 가입·등록을 유도하되, 포인트는 자사 서비스 내 한정 사용으로 운영.

### 마. LTV 확장 효과 추정

| 구분 | 단일 결혼 도메인 | **본 사업 (여정 전체)** |
|---|---|---|
| 평균 활성 기간 | 6~9개월 | **24~36개월** |
| 평균 결제 횟수 | 1~3회 | 5~10회 |
| 평균 LTV (추정) | 3.6만~7.2만원 | **12만~24만원** |

본 LTV 확장 메커니즘은 결혼 도메인의 본질적 한계를 극복하고, 동시에 신규 사용자 획득 비용(CAC) 대비 유리한 단위 경제를 형성한다.

## 5-3. 확장 전략 — 라이프 스냅 모듈 (검토 영역)

### 가. 시장 갭

스냅 촬영 시장은 결혼 외에도 기념일·가족·개인 프로필·임산부·영유아·반려동물 등 광범위한 사용처를 보유하나, 현재 사용자-작가 매칭은 인스타그램·네이버 카페·카카오톡 오픈채팅 등 분산된 채널에 의존하고 있다. **촬영 준비 과정(작가 비교·견적·소품·포즈 가이드)을 통합 지원하는 전용 플랫폼은 부재**하며, 본 사업은 결혼 후 사용자의 라이프스타일 스냅 수요를 통해 시장 검증을 시작한다.

2. **UX 우선순위 결정**: 어떤 페르소나의 페인을 우선 해결할지
3. **마케팅 메시지 차별화**: 페르소나별 광고·콘텐츠 톤 조정
4. **확장 가능성 평가**: 신규 페르소나 등장 시 시스템 대응 평가

> **보정 계획**: 본 페르소나는 가설 기반 초기 설정이며, 베타 단계에서 페르소나별 실제 사용자 5~10명 인터뷰로 보정한다. 분포 비중도 한국 결혼 시장 실측 데이터로 분기마다 재산정.

### 부록 7. 해자(Moat) 분석 — 시드 IR용

본 사업의 시간 누적 자산:

출시 직후 시점이므로 아래 해자는 **현재 일부 구축 + 향후 누적 예정** 자산을 포함한다. 시간 경과에 따라 후발 진입자가 따라잡기 어려워지는 구조다.

| 해자 | 현재 상태 | 향후 누적 메커니즘 | 후발 진입 장벽 |
|---|---|---|---|
| 도메인 데이터 | 업체 DB 초기 구축 | 후기·가격·이벤트 데이터 1년+ 누적 예정 | 시간 = 자산 |
| 사용자 데이터 락인 | 7개 차원 수집 구조 구축 | 사용자당 6~15개월 데이터 누적 예정 | 1명당 6~15개월 |
| 초개인화 정확도 | 페르소나 기반 초기 큐레이션 작동 | 사용자×데이터 누적 → 비선형 정확도 향상 | 비선형 성장 |
| 통합 플랫폼 전환 비용 | 정보·플래닝·커머스·B2B·AI 통합 구조 출시 완료 | 사용자별 통합 데이터 누적 | 통합 데이터 이전 불가 |
| 네트워크 효과 (커뮤니티) | 게시판·뱃지·포인트 구조 출시 | 사용자 누적 시 커뮤니티 가치 비선형 증가 | 사용자가 사용자를 끌어옴 |
| AI 프롬프트 튜닝 노하우 | 한국 웨딩 도메인 컨텍스트 구축 | 사용자 피드백 기반 지속 튜닝 | 도메인 지식 + 사용자 데이터 결합 |
