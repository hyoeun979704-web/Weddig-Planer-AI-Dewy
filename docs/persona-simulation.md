> 대상: 셀프웨딩 / 스몰웨딩 / 일반 웨딩 각 3인, 총 9인의 페르소나.

## 2. 시뮬레이션: 첫 방문~5번째 세션까지

### 공통 출발 — 첫 진입
1. WeddingBlessingSplash → 홈(`/`) → `HeroBanner` 캐러셀 자동 재생.
2. Bottom Nav: 스케줄·예산·홈·커뮤니티·마이페이지.
3. 로그인 시 `useWeddingInfoPrompt`가 작동해 `WeddingInfoSetupModal`이 자동으로 떠 결혼 정보·스타일·단계를 묻는다.

### A-1 이지원 (일반·호텔)
- **세션1 (15분)**: 모달에서 호텔 / 일반 / D-280 / "정보 알아보는 중" 입력. 체크리스트 32개 시드. 홈 둘러봄.
- **세션2 (20분)**: AI 플래너에서 "강남 호텔 1.5억 가능?" 질문. Quick Question(웨딩홀) → 추천 카드 5개. 찜 3개.
- **세션3 (25분)**: 예산 페이지에서 카테고리별 평균 비교, BudgetSplitSimulator 사용.
- **이탈 위험**: 홈이 generic — 본인의 D-280·진행률이 한눈에 안 보임. 매번 어디서 다음 액션을 잡을지 다시 탐색.

### B-1 윤채림 (스몰·한옥)
- **세션1 (15분)**: 모달 스타일 = small. 한복·예복·청첩장 카테고리 제외됨. 홈으로 돌아왔는데 여전히 "예복 추천" 등이 보임.
- **세션2~5 (각 40분)**: 인플루언서 페이지 깊이 탐색, 찜 다발 추가. 매거진. AI 스튜디오로 드레스 미리보기.
- **이탈 위험**: 매번 인플루언서 페이지 직접 진입 — 홈에서 그녀의 "감성 무드" 영감 피드가 안 뜸. 일주일 후 같은 콘텐츠 반복.

### C-2 권나라 (셀프·제주)
- **세션1 (20분)**: 모달 self. studio/dress_shop/makeup_shop 제외. DIY 5개 추가 일정 시드.
- **세션2~5 (각 60분+)**: 매거진 → DIY 부케 글 정독, 커뮤니티에 본인 글 작성, AI 스튜디오 드레스 시연.

## 5. UX 개선 방향 — 무엇을 / 왜 / 어떻게

### 5.1 [핵심] 결혼 스타일 인지형 홈 대시보드 (`PersonaDashboard`)
**문제**: 9인 모두에게 홈이 같음 — D-Day, 진행률, 다음 액션이 흩어져 있음.
**해결**:
- 로그인+결혼정보 입력 사용자에게는 홈 최상단에 통합 카드 노출:
  - D-Day · 진행률 (시드된 체크리스트 완료 비율)
  - "다음 액션 3개" — 이번 주 마감 임박 일정 (style별로 필터)
  - 데일리 스트릭 표시
- 셀프웨딩 → DIY 미션 표시, 스몰웨딩 → 큐레이션 영감, 일반 → 양가 분담·견적.
**기대 효과**: 매 세션 진입 후 "지금 뭘 해야 하지" 결정 비용 제거 → 1세션 5~10분 증가.

### 5.2 데일리 체크인 스트릭 + 미션 (`useDailyStreak`)
**문제**: 권나라 같은 헤비유저에게 보상 없음. 장유진 같은 라이트유저에게 재방문 동기 약함.
**해결**:
- 매일 첫 방문 시 자동 체크인 → 연속 N일 표시.
- 스타일별 3개 데일리 미션 (예: 셀프 → "오늘의 DIY 매거진 1개 읽기", 일반 → "양가 분담 시뮬레이션 한 번 돌리기").
- 미션 완료 시 하트(포인트) 적립과 시각적 보상.
**기대 효과**: 주당 세션 수 5.5 → 7.0, 셀프웨딩 권나라 유형 리텐션 강화.

### 5.3 세션 타이머 + "오늘 N분 사용" 표시 (`useSessionTimer`)
**문제**: 사용 시간 데이터 자체가 없어 향후 개선 계측이 안 됨.
**해결**:
- localStorage 기반 일일 누적 세션 시간 기록 (탭 active/hidden 이벤트 기반).
- 마이페이지 / 대시보드에 표시 → 셀프 정량화 효과.
- 향후 Supabase로 보내 코호트 분석.
**기대 효과**: 자기 인식 → 평균 체류 시간 자연 증가, A/B 계측 기반 확보.

### 5.4 스타일별 AI 플래너 Welcome 카드
**문제**: AI 플래너 진입 시 모두에게 같은 4개 quick question.
**해결**:
- weddingStyle에 따라 quick question 슬롯 일부를 교체:
  - self → "셀프촬영 로케이션 추천", "DIY 부케 가이드"
  - small → "스몰 베뉴 추천", "답례품 아이디어"
  - general → "양가 분담 시뮬", "호텔 vs 컨벤션 비교"
**기대 효과**: AI 플래너 첫 메시지 전송률 상승 → 평균 대화 길이 증가.

### 5.5 진행률 시각화 (Progress Ring)
**문제**: 시드된 체크리스트가 32~37개인데 어디까지 왔는지 체감이 없음.
**해결**:
- 홈 대시보드 카드에 도넛형 진행률 + 단계 표시 (`just_started` → `wrapping_up`).
- 단계 변경 시 축하 토스트.
**기대 효과**: 게이미피케이션 효과로 세션당 액션 수 증가.

## 6. 측정 계획 (Definition of Done)

배포 후 2주 관찰 지표:
1. **세션당 평균 분(分)**: 30분 → 42분.
2. **DAU/MAU**: 25% → 35%.
3. **AI Planner 첫 메시지 전송률**: 28% → 45%.
4. **체크리스트 완료율 (시드된 아이템 중 완료 %)**: 12% → 22%.
5. **결제 전환 (Premium)**: 1.8% → 2.5%.
