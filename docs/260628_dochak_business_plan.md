## □ Ⅱ. 문제 인식 (Problem)

### ○ 1. 추진 배경 및 필요성

**가. 시장 배경**

국내 결혼 준비는 1인당 평균 6~12개월의 기간과 3,000만~5,000만원의 비용이 소요되는 대표적 고비용·고관여
의사결정 영역임(통계청 인구동향조사 및 결혼정보업체 자료). 그러나 ① 결혼 준비에 수반되는 비용·정보 부담
자체가 결혼의 지연·기피 요인으로 작용하고 있으며(국토연구원 : 주거·결혼 비용 부담이 혼인·출산율에 부정적
영향), ② 기존 플랫폼(웨딩북·아이웨딩·요즘웨딩 등)은 '표준 결혼식' 이용자를 전제로 동일 정보를 일괄
제공함에 따라 비표준 사례 및 정보 과부하 이용자층이 충분히 대응되지 못하고 있음.

**나. 해결 과제(Pain Point)**

| 핵심 문제 | 영향 이용자 | 본 사업의 해결 방안 |
|---|---|---|
| 정보 과다로 인한 의사결정 피로 | 표준 준비자(약 50%) | 20개 페르소나 기반 필요 정보 선별 제공 |
| 비표준 사례 가이드 부재 | 스몰·셀프·노웨딩(약 30%) | 상황별 전용 페르소나·체크리스트 |
| 개별 상황 특수성 미대응 | 임신·재혼·국제·1인 진행(약 15%) | 임신·재혼·국제결혼 전용 모드 |
| 지역·예산별 맞춤 정보 부재 | 지방 거주자·저예산층 | 16개 지역 × 7개 항목 평균가 + 업체 큐레이션 |
| 고가 계약 전 결과물 확인 불가 | 전(全) 이용자 | AI 이미지 생성(드레스·메이크업·헤어·촬영) |

**다. 환경 변화에 따른 기회**

- (기술) 거대언어모델·이미지 생성모델이 상용 수준에 도달하였으나, **결혼 도메인에 특화하고 답변 신뢰성을
  검증한 서비스는 부재**한 상황임.
- (시장) 2024년 혼인건수 **222천 건(전년 대비 +14.8%)으로 13년 만에 두 자릿수 증가**하였고, 미혼자 결혼
  의향은 **62.2%(2021년 대비 +11.4%p)**, 25~29세 여성은 57.4%(+9.2%p)로 상승하여 **결혼 수요가 회복
  국면**에 진입함(통계청·관계부처).
- (소비행태) 스몰·셀프웨딩이 2015년 대비 3~4배 증가하여 표준형 정보의 효용이 저하되고 초개인화 수요가 증대됨.

- **확장 고객** : 결혼 완료 후 신혼·기념일·육아 등 라이프스테이지로 자연 이동(고객생애가치 확장).

## □ Ⅲ. 실현 가능성 (Solution)

### ○ 1. 개발 현황 — 출시·실증 완료 단계

본 사업은 기획 단계가 아니라 **정식 출시 후 실데이터로 검증된 단계**에서 도약을 신청함(2026. 05. 22. 출시).
아래 지표는 2026. 06. 28. 기준 코드 및 운영 데이터베이스 실측값임.

**가. 제품 규모**

| 영역 | 구축 현황 |
|---|---|
| 서비스 채널 | 웹·PWA(dewy-wedding.com) 및 Android 네이티브 앱 정식 운영 |
| 화면·구조 | 화면 183개 / 라우트 135개 / DB 테이블 147개 |
| AI 기능 | 서버리스 함수 60여 종(AI 약 20종), L1~L5 환각차단 RAG 운영 |
| AI 시뮬레이션 | 6종 운영(웨딩컨설팅·스드메 통합·드레스·메이크업·헤어·종이청첩장), 3종 개발 예정 |
| 결제 | 웹 카카오페이 + 네이티브 인앱결제(애플·구글) 코드 완비, 영수증 검증·멱등 처리 |
| B2B | 사업자 콘솔(가입→인증→등록→검수→정산) 완비, 사업자 전용 앱 출시 계획 확정 |
| 기반 | 인증·권한 3계층(이용자·사업자·운영자), 검색최적화(SSR), 도메인 경계 자동 검증 |

| 구 분 | 지 표 | 내 용 |
|---|---|---|

| | 콘텐츠 **16,131건**, 영상·채널 2,300여 건 | 검색최적화·AI 근거 데이터 |

> 데이터 자산·유료 결제·AI 실사용이 모두 확인되어, 사업의 핵심 가설(제품·데이터·수익)이 검증된 상태임.
> 본 지원금은 미검증 가설이 아니라 **검증된 전환 구조의 확대(마케팅·인력)**에 투입되어 자금 효율이 높음.

### ○ 2. 핵심 기술 자산 (기술성·혁신성)

1. **L1~L5 다층 환각차단 RAG 아키텍처(자체 설계)** : ① 의도 분류 후 DB 직접응답(거대언어모델 미호출),
   ② 16개 지역 × 7개 항목 평균가·실업체 데이터 근거 주입, ③ 모든 금액에 출처태그 강제, ④ 출력 후처리
   감사, ⑤ 이용자 메모리 검증 절차로 구성됨. 답변의 사실성을 구조적으로 확보하여 금융·의료 등 신뢰성
   중심 도메인으로 이전 가능한 핵심 지식재산임.
2. **20개 모드 페르소나 초개인화 엔진** : 임신·국제결혼·재혼·스몰·셀프·1인 진행 등 상황별 및 성향별
   분류를 결정적·멱등 방식으로 수행하며, 단위테스트로 정합성을 검증함.
3. **멀티모달 이미지 생성 파이프라인(6종)** : 이용자 사진과 참조 이미지를 합성하여 결과물을 사전 제시.
   전자상거래 가상 피팅 등으로 확장 가능함.
4. **머신러닝 분류 모델(KLUE-RoBERTa)** : 콘텐츠 토픽 자동 분류(학습 완료, 배포 예정).
5. **멀티프로바이더 추상화 및 평가 인프라** : 모델 교체 용이성 확보, 프롬프트 캐싱(입력비용 약 50% 절감),
   품질·비용 자동 평가 체계 운영.
