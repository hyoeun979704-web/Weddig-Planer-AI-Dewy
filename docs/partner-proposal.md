## 왜 Dewy 파트너인가요?

- **이미 36개 업체가 함께합니다**: 정식 출시 후 36개 웨딩 업체가 Dewy에 입점해 고객과 만나고 있습니다.
- **구매 의향이 뚜렷한 고객**: Dewy 사용자는 지금 결혼을 준비하며 업체를 찾는 예비부부입니다.
- **개인화 매칭**: 지역·예산·페르소나에 맞춰 노출돼, 막연한 광고가 아니라 **맞는 고객**과 만납니다.
- **앱 안에서 완결**: 노출 → 문의 → 견적 → 예약까지 한 흐름. 고객을 외부로 흘리지 않습니다.

## 입점하면 쓸 수 있는 기능 (업체 포털)

- **업체 정보·갤러리 관리** — 상세페이지에 우리 업체를 직접 꾸미기
- **쿠폰 발행** — 검토 없이 즉시 노출되는 할인 쿠폰으로 고객 유인
- **이벤트·상품 등록** — 시즌 프로모션·패키지 노출
- **받은 견적 요청(리드)** — 조건에 맞는 고객의 견적 요청에 응답 → 수락 시 연락처 공개·인앱 채팅 연결
- **문의/예약 관리** — 고객 문의 확인·답변
- **고객 후기·통계** — 조회수·찜·후기·평점 확인

> 📘 입점 후 사용법은 **「업체 사장님 가이드」**(가입→제휴→등록→대시보드, 실제 화면 포함)로 안내드립니다.
