## 3. 기능별 페인포인트 & 개선 (요약)

> 이 PR에서는 (T) 표시된 튜토리얼 항목만 실제 구현. 나머지는 백로그.

### 3.1 튜토리얼 (T·이번 PR 본 영역)
- **현재 페인포인트**:
  - `useTutorial.ts`의 `APP_TOUR_STEPS`가 `nav-ai` 셀렉터를 참조하지만 BottomNav에는 AI 탭이 없음 → 빈 강조 박스, 툴팁만 떠 있음.
  - AIPlanner 페이지의 가이드는 `ai-header`/`ai-suggestions`/`ai-input` 셀렉터를 참조하지만 페이지에 해당 attr이 **하나도 없음** → 3단계 모두 빈 화면 코치마크.
  - 가이드 목록이 평평한 리스트 (`FEATURE_GUIDES`). 어디서 시작해야 하는지 직관이 없음.
  - 결혼 스타일 무관하게 모든 가이드 노출 → 셀프웨딩 사용자에게 스드메 가이드.
  - 완료해도 다음에 무엇을 해야 할지 가이드 없음.
- **개선 (이번 PR)**:
  1. 챕터 구조 도입 (`tutorialChapters.ts`) — 기초 / AI / 예산·일정 / 커뮤니티·커플 / 프리미엄.
  2. 스타일 인지 필터 — 각 챕터/레슨에 `requiresStyles?` 옵션. 셀프웨딩에서 스드메 챕터 자동 제외.
  3. 진행률 추적 — `useTutorialProgress`가 localStorage에 레슨 완료를 저장.
  4. 새 튜토리얼 페이지 — 진행률 링 + 챕터 카드 + "이어서 시작" 버튼.
  5. 첫 방문 시 자동 노출되는 환영 시트(`TutorialWelcomeSheet`) — 결혼 정보 입력 완료 직후 1회.
  6. 깨진 셀렉터 수정 — 잘못된 `nav-ai` 단계 제거, AIPlanner에 `data-tutorial` attr 추가.
  7. PersonaDashboard에 튜토리얼 진행률 노출 — 미완 챕터 1개를 우측 상단 작은 칩으로.

### 3.2 AI 플래너 (백로그)
- 페인: 첫 질문 전송률 낮음 / 같은 질문 반복 시 컨텍스트 손실.
- 개선: 최근 질문 기록 표시, 컨버세이션 저장 옵션.

### 3.3 AI 스튜디오 (백로그)
- 페인: B1·C1·C2가 핵심인데 결과 공유·저장 동선 길음.
- 개선: 결과 → 커뮤니티 글쓰기 1탭 공유.

### 3.4 스케줄 / 예산 (직전 PR에서 이미 스타일 제외 처리됨)
- 페인: 새 일정 추가 UX가 무겁다.
- 개선: 빠른 메모 위주 카드 추가.

### 3.5 커뮤니티 (백로그)
- 페인: 글쓰기 진입 후 사진 업로드 동선이 분산.
- 개선: 사진 우선 입력 패턴.

### 3.6 프리미엄 PDF (백로그)
- 페인: 결제 동선이 안내 PDF 옆에 같이 있어 헷갈림.
- 개선: 미리보기 → 결제 분리.

### 3.7 포인트·하트 (백로그)
- 페인: 적립 룰이 흩어져 있어 무엇을 하면 받는지 불투명.
- 개선: 마이페이지에 "오늘의 적립 방법 3개" 카드.
