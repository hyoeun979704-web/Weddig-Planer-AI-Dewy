## 1. 브랜드 사실 (틀리면 안 됨)

- **앱명:** Dewy(듀이) — "AI 웨딩플래너와 함께하는 결혼 준비"
- **채널:** 웹 + 안드로이드/PWA · **`dewy-wedding.com`**
- **요금:** 프리미엄 구독 **₩4,900/월** + 하트(인앱결제)
- **페르소나:** 일반 / 스몰 / 셀프웨딩 분리 지원
- **차별 기능:** 방구석 드레스 투어 · AI 메이크업 시뮬 · AI 챗봇 플래너 · 무료 모바일/종이 청첩장 · 꽃 머지 미니게임
- **신규 가입 공통 혜택(이미 운영 중):** **1개월 프리미엄 무료 + 보너스 하트** → 이건 **전원** 혜택이라 "테스터 전용"으로 광고하지 않는다.
- **앱 아이콘 = 핵심 비주얼 모티프:** pink→coral→blue 그라데이션 하트(광택 3D).

## 2. 확정 이벤트 스펙 (모든 소재의 사실 출처)

### A. 베타 테스터 모집 🤖
- **두 트랙으로 분리해 표기**(혼동 방지):
  - **[테스터] — 안드로이드 한정.** 충분히 사용 + **설문 참여**.
    - 혜택: (신규 공통)1개월 구독 + 하트 50 **＋ 테스터 전용: 정식 출시 후 구독 2개월 연장 + '1호 테스터' 뱃지(앱내 영구 표시)**
  - **[후기] — 누구나(아이폰·웹 포함).** **SNS 후기만 보상 → 하트 +50** (✅ 결정: 플레이스토어 후기는 보상 대상 아님 — 구글 정책 리스크 회피. 플스토어 후기는 무보상 자발 권유, `05-compliance-ops.md` §1).
- **신청:** 모집 글에 ❤️ + **DM으로 이메일** 전송 → 등록.
- **개인정보 고지(필수):** "이메일은 **베타 등록 용도로만** 쓰고 **종료 후 파기**합니다."
- **상태:** 즉시 집행 가능.

### B. 부케 20만원 할인권 💐
- **참여:** 꽃 머지 미니게임 **최종 단계 도달** → 결과 화면 캡처 → **SNS 업로드 + 오피셜 계정(@dewy_ai_weddig_planer) 태그**(공개 계정 한정).
- **보상:** 제휴 부케업체 **20만원 할인권**(앱내 쿠폰으로 지급, 실물 배송 아님).
- **검증:** 게임 점수 **서버 기록과 교차 확인**(스크린샷 단독 신뢰 금지) + 오피셜 태그 필수.
- **규모(확정):** **선착순 100명** · 1인 1매.
- **제휴(확정):** 제휴 부케업체 **2곳 확정**(추가 가능성 있음). 업체명·할인 적용 조건은 `05-compliance-ops.md` §3에 따라 안내에 명시.

> ✅ **오피셜 계정 핸들:** `@dewy_ai_weddig_planer` (확정). 모든 SNS 소재·인증 태그에 이 핸들 사용.

## 3. 비주얼·톤 규칙 (전 소재 공통)

- **컬러:** blush-pink → ivory/cream 배경 + pink-coral-blue 그라데이션 하트 포인트. 골드 얇은 액센트 허용.
- **톤:** 따뜻·다정·예비부부 공감. **과장·허위 효능 금지**(SusVibes·채널정책). 첫 1초/첫 줄 후킹 + 브랜드명 노출.
- **이미지 생성:** GPT/이미지 모델 프롬프트는 **영문 + "no text/letters/numbers"**로 받아 **한글 카피는 편집에서 오버레이**(다국어·오타 방지).
- **규격 표준:** 세로 소재 4:5 또는 9:16, 유튜브 16:9. 텍스트 안전 여백 상·하 확보.

## 4. 측정·채널 매핑 (원소스 멀티유즈)

**UTM 규칙 (모든 외부 링크에 부착 — 성과 측정의 전제):**
```
?utm_source={threads|instagram|youtube|naver_clip}
&utm_medium={organic|cpc}
&utm_campaign=beta_bouquet_2026
&utm_content={popup|cardnews|youtube|reels}
```

**소재 × 채널 (1개 제작 → 다채널 재활용):**

| 소재 | 1차 채널 | 재활용처 |
|---|---|---|
| 앱내 팝업/배너 | 앱 홈(진입 팝업·띠배너) | — |
| 카드뉴스 | 인스타 피드 · 스레드 | 네이버 블로그 삽화 |
| 유튜브 15초 | 유튜브 인스트림 · 쇼츠 | 릴스 · 네이버 클립 |
| 릴스 12초 | 인스타 릴스 | 쇼츠 · 네이버 클립 |

**KPI (성공 기준):**
- **베타:** 테스터 신청 수 · 설문 완료율 · **SNS 후기 수**(보상 대상) · 플스토어 후기 수(무보상) · **자발적 공유율 15%↑** · **사전등록 30명↑**(business-plan 목표 연계).
- **부케:** 게임 최종단계 도달 수 · SNS 인증 수 · 할인권 발급/사용 수 · 오피셜 태그 도달(노출).
- **광고(유튜브):** 소재별 CTR · **CPI** · 가입 전환 → 48h 후 승자 소재 집중.

> ⚠️ **지어낸 수치 금지:** "1000명"은 모집 커뮤니티 이름(스하리1000명프로젝트)일 뿐 **테스터 정원이 아니다.** 정원·할인권 규모 등 숫자는 §2 확정값 또는 "가안" 표기만 사용.
