## TL;DR (핵심 성과)
- **일정**: 표준 5단계가 전 페르소나 동일하던 것 → 페르소나 델타(임신 차수·소형 형식/규모·재혼·셀프·
  노웨딩 등) + 리서치 보강(상견례·본식 영상·가봉·하객 인원). **표시 전용**(adopt 방식)이라 회귀 0.

  흩어진 부호 계산을 **단일 헬퍼 `netManwon`** 으로 통일(드리프트 차단). 잔여 인라인 0(grep 전수).

- ✅ 환불 입력은 클라 토글이지만 합계는 **서버 데이터 재집계**(클라가 보낸 합계 신뢰 안 함).
- ✅ PII·시크릿 신규 노출 없음. 위젯 스냅샷은 여전히 토큰/PII 미포함(`92742f3` 유지).
- ✅ 프로덕션 마이그(`20260622140000`)는 `add column if not exists … default false` — 멱등·무중단,
  **사용자 명시 승인 후** MCP 적용(컬럼 존재 SQL 검증 완료).
