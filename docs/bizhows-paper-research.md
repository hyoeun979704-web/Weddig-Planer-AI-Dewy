# 비즈하우스(BizHows) 종이 청첩장 전수조사 (PAPER 레퍼런스)

> 종이 청첩장 기준점 = **비즈하우스**. 실제 제품 라인·규격·용지·후가공·에디터·구성품을 조사해
> 우리 종이 트랙(`docs/paper-invitation-formats-plan.md`)의 기준으로 삼는다.
> ⚠️ 출처 한계: bizhows.com / help.bizhows.com 는 봇 직접 fetch 차단(403) → **웹검색 스니펫 +
> 보조 출처**로 정리. 제품별 정확한 mm·옵션은 라이브 상품 페이지/파트너 확인으로 최종 검증 필요.

## 0. 한 줄 요약 — 비즈하우스의 본질

**비즈하우스는 "셀프디자인 인쇄 플랫폼"**이다. 강점은 ① **카테고리별 규격·사이즈 자동 프리셋**(사용자가
mm 안 만짐) ② **풍부한 용지·후가공 옵션** ③ **무료 템플릿 + 셀프 편집기 + 장바구니 미리보기**
④ **무료 용지 샘플** ⑤ **웨딩 SUITE**(청첩장 단품이 아니라 봉투·식권·포스터까지 한 세트).
바람개비·여권 같은 기상천외 형태는 "제품"이 아니라 **도무송(다이컷) 발주**로 푸는 영역.

## 1. 제품 라인업 (청첩장/웨딩포스터/봉투 카테고리)

- **1단 엽서카드(청첩장)** — 단면/양면 플랫 카드.
- **2단 엽서카드 / 2단 청첩장** — **접이식(접어서 발송)**, 양면.
- **2단 미니 청첩장(2025)** — 소형 2단.
- **프리미엄 고평량 엽서 / 프리미엄 청첩장** — 고평량 프리미엄 용지 라인.
- **웨딩포스터(단면 포스터)** — 식장 입구용 대형.
- **봉투** — 엽서봉투, 트레싱지 봉투.
- **식권(meal ticket)** — **명함 정도 사이즈**, 단면, 라운딩 옵션.
- **부가**: 스티커·실링왁스 스티커·포토테이블·현수막·어깨띠.

→ 핵심: 청첩장은 **1단 / 2단(접이식)** 두 축이 메인. 나머지는 **세트 구성품**(봉투·식권·포스터).

## 2. 규격 (mm)

- **엽서 기본**: `100×150` 또는 `150×100`(방향 선택). (A6 근사)
- **봉투**: `165×115`, `180×120`, `185×105`, `195×135` — **1단·2단 엽서카드 모두 호환**.
- **식권**: ≈ 명함 사이즈(약 `90×50`).
- **2단**: 펼침면을 가운데 접어 발송(완성 = 펼침의 1/2 폭).
- 카테고리별 사이즈가 **자동 설정** → 사용자는 규격 고민 없이 디자인만.

## 3. 용지 (paper)

- **청첩장 추천**: **랑데뷰 240g / 310g**.
- 기타: **스노우(무광)·아트지·몽블랑 네츄럴(고급, 펄프결·질감)**.
- **무료 용지 샘플 제공** — 실물 만져보고 선택 권장.

## 4. 후가공 (finishing)

- **박**: 금박·은박·홀로그램박.
- **UV 코팅**: 부분 광택(도드라짐).
- **형압**: 입체(눌러서 촉감).
- **도무송(다이컷)**: 발주 가능(커스텀 형태·모서리).
- **라운딩**: 모서리 둥글림.
- **양면** 인쇄.
- 가이드: 너무 작은 글씨·얇은 선은 박/형압 효과 약함 → 굵고 여백 명확한 디자인 권장. 장바구니
  미리보기 가능하나 실물과 차이 있을 수 있음.

> **우리 파트너 capability 매핑**: 파트너 = **레이저 컷팅 + 코팅** 가능 → 비즈하우스의 **도무송
> (다이컷)·라운딩·코팅**에 대응. **박·형압은 비즈하우스엔 있으나 우리 파트너 미확인**(별도 장비).

## 5. 셀프디자인 에디터 / 주문 플로우

- **업종/상황별 무료 템플릿** 선택 → **무료 편집기로 문구 수정**(텍스트 중심 편집) → 장바구니
  **미리보기** → 주문. (명함 셀프디자인 가이드가 대표 예시 — 청첩장도 동일 플로우.)
- 규격은 상품 카테고리가 자동 결정 → 편집기는 "안전영역 안에서 콘텐츠 채우기"에 집중.

## 6. 우리 종이 트랙에 주는 시사점 (계획 델타)

`docs/paper-invitation-formats-plan.md` 를 아래로 보강한다:

1. **단품이 아니라 SUITE 로 설계**: 청첩장(1단/2단) + **봉투** + **식권** + **웨딩포스터** 를
   한 세트 포맷군으로. (봉투-카드 사이즈 호환 매트릭스 포함.)
2. **포맷·규격 자동 프리셋**: 사용자가 mm 를 만지지 않게 포맷 선택 시 규격·세이프·재단 자동.
   (우리 `InvitationPrintSpec` 프리셋 시드로.)
3. **용지·후가공 옵션 모델**: `PrintFinishing` 에 **용지(랑데뷰/스노우/몽블랑)·평량(240/310)·
   코팅·라운딩·도무송** 을 셀렉트로. 박/형압은 "파트너 확인 시 노출"로 플래그.
4. **무료 샘플·미리보기**: 장바구니 미리보기 + (가능하면) 용지 샘플 안내 UX.
5. **메인 포맷 우선순위 재정렬**: 비즈하우스 기준 **1단 엽서·2단 접이식**이 메인 → **이 둘을 최우선**.
   (앞서 단면 엽서/포토카드부터로 잡았는데, **2단 접이식**이 시중 수요의 핵심이므로 fold 모델을
   앞당길 가치 있음.) 바람개비/여권/초콜릿 = 도무송 기반 부티크 라인(후순위).
6. **봉투 사이즈 호환 매트릭스**: 165×115·180×120·185×105·195×135 ↔ 1단/2단 카드 매핑 데이터.

## 7. 미확인 / 후속 확인 필요

- 제품별 **정확한 완성 규격(mm)**: 2단 청첩장·2단 미니·프리미엄 라인의 펼침/완성 치수(라이브 페이지 차단으로 미확보).
- **셀프 편집기 상세 UX**(텍스트 외 사진·요소 편집 범위, 템플릿 수).
- **파트너 capability 확정**: 박·형압 가능 여부, 도무송 입고 사양(라인 색/두께/포맷), 용지 라인업.
