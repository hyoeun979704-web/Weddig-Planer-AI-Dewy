## 2. "설마 이것까지?" 함정 상세 (놓치기 쉬움)
- **Apple 프라이버시 매니페스트**(2024-05~): 앱 `PrivacyInfo.xcprivacy` + **AdMob 등 SDK** 매니페스트
  필수. required-reason API(UserDefaults·파일타임스탬프 등) 사유 선언. 없으면 반려. → Capacitor/플러그인
  최신화 + 앱 매니페스트 작성.
- **Google 신규 개인계정 테스트**: 2023-11-13 이후 **개인** 계정은 프로덕션 전 **12 테스터 14일 연속**
  비공개테스트 필수(법인계정 면제). → **출시 2주+ 선행** 필요. **계정 유형 확인이 최우선.**
- **SiwA 토큰 폐기**(TN3194): SiwA 도입 시 계정삭제에서 `appleid.apple.com/auth/revoke` 호출 의무.
  → `delete-account` 엣지함수에 Apple 토큰 폐기 추가(iOS 한정).
- **암호화 수출 신고**: `Info.plist`에 `ITSAppUsesNonExemptEncryption=false`(표준 HTTPS만 쓰면) 추가 →
  매 업로드 설문 생략. 없으면 빌드마다 질문.
- **Apple 4.2 최소기능**: Capacitor 앱이 "그냥 웹사이트"로 보이면 반려. 네이티브 기능(푸시·결제·공유 등)·
  네이티브 UX 충분히. → 네이티브 통합 포인트 확보.
- **콘텐츠 등급**: Google IARC 설문 + Apple age rating 설문(앱이 UGC·AI 포함이면 정확히).
- **위치정보법**: 지오로케이션으로 "내 주변 업체" 등 **개인위치정보 이용 시 위치기반서비스사업 신고** +
  약관 동의(이미 약관 보유). 현재 코드는 지오로케이션 미사용 → **기능 켜기 전** 신고.
- **야간 광고 수신**: 21~08시 광고성 정보(앱푸시/SMS) 발송엔 **별도 야간 수신 동의** 필요(정보통신망법).

## 3. 우선순위 액션 플랜
### P0 — 출시 차단 (반드시 선행)
1. **심사용 데모 계정**: 고정 리뷰어 계정(하트·프리미엄 충전 상태) + 제출노트에 자격 기재.
   (옵션: 리뷰어용 우회 결제/프리미엄 토글.)
2. **결제 IAP 분리**(`260620_payment_compliance_plan.md` P0-A) + **G1 SiwA·G2 ATT·G3 구독고지**.
3. **Apple 프라이버시 매니페스트**(앱 + AdMob/Capacitor SDK 최신) — iOS.
4. **AI 생성물 인앱 신고**(genAI 정책) — 피팅·스드메·촬영·청첩장 결과에 신고(ReportDialog 재사용)+모더레이션.
5. **Google 계정유형 확인** → 신규 개인계정이면 **비공개테스트 12×14일 즉시 착수**(출시 일정 역산).

### P1 — 출시 전 권장
6. **쿠키/추적 동의 배너(웹)** — 분석/AdSense 로드 전 opt-in.
7. **푸시 권한 플로우**(쓸 거면) — `@capacitor/push-notifications` + 동의 + 발송 백엔드. 안 쓰면 인박스만 유지(정상).
8. **야간 마케팅 동의** 분리 + **위치사업 신고**(위치기능 켤 때).
9. **SiwA 토큰 폐기**를 `delete-account`에 추가(iOS).
10. **암호화 수출 신고 plist 키**, **콘텐츠 등급 설문**, **Data Safety/App Privacy 폼**.
11. **환불 요청 UI**(현재 문의로 대체 — 정책상 OK이나 UX 보강 권장).

### P2 — 출시 후/차기
12. 타깃 API 상향, 딥페이크 탐지(현재 동의 기반), 접근성/성능.
