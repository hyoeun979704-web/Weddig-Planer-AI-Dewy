## 3. 연령 등급 (IARC) 설문 답변

Play Console "앱 콘텐츠 → 등급" 메뉴의 설문 표준 답변. **카테고리: 참고용**.
설문은 IARC가 운영하며 항목 표현이 바뀔 수 있어, 동일 취지로 답변.

| 항목 | 답변 | 근거 |
|---|---|---|
| **카테고리** | 참조 (Reference / 라이프스타일 / 결혼·웨딩) | Play Console 권장 |
| 폭력 (Violence) | 아니오 | 어떤 형태의 폭력 묘사도 없음 |
| 성적 콘텐츠 (Sexuality) | 아니오 | 신혼 관련 단어는 사용하지만 성인 콘텐츠 미포함 |
| 노출 (Nudity) | 아니오 | 없음 |
| 폭언·욕설 (Profanity) | 아니오 | 앱 자체엔 없음. 커뮤니티 UGC는 별도 모더레이션 정책 |
| 차별·혐오 | 아니오 | 없음 |
| 도박·카지노 | 아니오 | 없음. 포인트는 상품 결제용 |
| 약물·마약 | 아니오 | 없음 |
| 담배 | 아니오 | 없음 |
| 알코올 | 아니오 | 웨딩홀 옵션 정보에 "주류 반입 가능" 표기 정도. 음용 묘사 없음 |
| **사용자 생성 콘텐츠 (UGC)** | **예** | 커뮤니티 게시글·댓글·사진 업로드 |
| UGC 모더레이션 정책 보유 | **예** | 커뮤니티 가이드라인 + 신고·삭제 기능 (출시 전 점검 필요) |
| **사용자 간 상호작용** | **예** | 댓글, 좋아요, 커플 연동, 채팅 없음 |
| 위치 공유 | 예 (앱 내부) | 결혼식장·스튜디오 지역 기반 검색. 사용자 간 위치 공유는 없음 |
| **앱 내 결제 (Digital purchase)** | **예** | 하트 충전, 프리미엄 구독, 외주 결제 (Toss / 카카오페이) |
| **개인정보 수집** | **예** | 이메일·전화번호·결혼 관련 정보·사진 |
| 광고 노출 | 아니오 (외부 광고 SDK 미포함) | 자체 제휴 콘텐츠는 있으나 외부 광고 네트워크 없음 |

**예상 등급**: **만 12세 이상** (Teen / PEGI 12 / IARC 12+)
- UGC + 결제 + 위치 기반 검색 → 12세 미만 권장 등급 어려움
- 폭력·성적 콘텐츠 없음 → 17+ 등급 사유 없음
