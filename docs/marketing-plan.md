> **실행 파이프라인**(SSOT→변환→배포)은 `content-distribution.md`, **주제·콘텐츠 페르소나**는 `wedding-intel.md` 참조.

## 0. 공통 토대 (모든 채널 실행 전 먼저 세팅 — 0.5일)

- **포지셔닝 한 문장:** "예비부부의 결혼 준비, AI랑 둘이 같이 끝낸다 — Dewy"
- **3대 메시지 축** (모든 콘텐츠는 이 중 하나에 귀속):
  1. **편의** — 체크리스트·예산·스드메·청첩장 한 앱에서 (흩어진 준비를 통합)
  2. **AI 차별점** — 방구석 드레스 투어/메이크업 시뮬, AI 맞춤 조언·견적 PDF (경쟁 앱엔 없음)
  3. **페르소나 공감** — 일반/스몰/셀프웨딩까지 분리 지원 (대부분 앱은 일반웨딩만)
- **추적 세팅 (설치 KPI 측정의 전제):**
  - 모든 외부 링크에 **UTM 파라미터** + 단일 랜딩(`dewy-wedding.com/?utm_source=naver_blog...`)
  - 채널 구분: `utm_source`(naver_blog / naver_cafe / naver_clip / youtube / meta), `utm_medium`(organic / cpc), `utm_campaign`
  - 가능하면 **딥링크 1개**(플레이스토어 + 웹앱 분기)로 통일
  - GA4 / 내장 `src/lib/behavioralSignals.ts`에서 소스별 가입·전환 확인
- **재활용 원본 6종**(이걸로 전 채널 파생):
  1. 드레스 투어 before/after 영상
  2. AI 챗봇 상담 대화 캡처
  3. 청첩장 제작 타임랩스
  4. 예산표/체크리스트 화면
  5. 스몰/셀프웨딩 가이드
  6. 창업 스토리

## 1. 네이버 블로그 — SEO 자산 (장기 인지도의 기둥)

예비부부는 결혼 준비를 네이버 검색으로 시작함. 블로그는 즉효는 낮지만 **검색 유입 자산**이 누적되는 유일한 채널.

**운영 강도(현실):** 주 1~2건, 회당 1,500자+ / 이미지 8장+ (앱 스크린샷 재활용)

**전략:**
- **정보성 70% : 앱 소개 30%** (네이버는 노골적 광고 글에 페널티 → 정보 먼저, 앱은 "이 과정 Dewy로 자동화됨"으로 자연 삽입)
- **롱테일 키워드 공략** (대형 키워드는 업체 광고글에 밀림):
  - "2026 결혼 준비 순서 체크리스트", "스몰웨딩 준비 비용 정리", "셀프 청첩장 만들기 무료",
    "웨딩 예산 평균 (지역별)", "방구석 드레스 투어 후기", "예비부부 가계부 나누기"
- **시리즈 연재:** "결혼 준비 D-200 ~ D-day 월별 가이드" (앱 일정 기능과 1:1 매칭)
- 각 글 하단 **고정 CTA 배너**: "이 체크리스트, Dewy 앱에서 자동 관리 → [링크]"
- **상위노출 보조:** 네이버 인플루언서/이웃 교류, 발행 시간대 일관, 대표 키워드 제목 앞단 배치

**첫 4주 산출물 우선순위:** ①결혼 준비 순서 총정리 ②스몰웨딩 비용 ③셀프 청첩장 무료 만들기 ④AI 드레스 투어 후기

## 2. 네이버 카페 — 타겟 밀집지 (초기 유저·후기의 금광)

예비부부가 가장 활발히 모이는 곳. 노골적 홍보 = 강퇴/차단 → "도움 주는 예비신부/신랑" 포지션으로 신뢰부터.

**핵심 카페:** 다이렉트결혼준비, 대형 결혼준비 카페, 지역 맘카페/예신카페, 스몰/셀프웨딩 전용 카페

**전략 (2단계):**
- **1단계 (신뢰 적립, 2~3주):** 홍보 0%. 질문 답변·후기 공유로 등급 상승. "저 이거 이렇게 했어요" 식 경험 공유.
- **2단계 (자연 노출):** 답변 끝에 "저는 Dewy 앱으로 예산 나눠 관리했는데 편했어요" 식 **맥락 삽입** + 규정 허용 시 후기 게시
- **무료 가치 미끼:** "결혼 준비 체크리스트 PDF 무료로 드려요"(앱 PDF export 활용) → 댓글/쪽지로 앱 링크 = 자연 유입
- **베타 모집 활용:** "베타 테스터 모집 + 하트 무료 지급" 이벤트로 초기 유저 + 후기 동시 확보 (소프트런치 단계 최적)
- **공식 제휴:** 카페 운영진에 배너/제휴 문의 (유료지만 타겟 정확도 최고 — 예산 풀리면 우선순위)

**주의:** 1인 다중 계정 자작 후기는 적발 시 브랜드 치명타 → **진짜 베타 유저 후기**로만.

## 3. 네이버 클립 — 숏폼 인지도 (네이버 생태계 노출 부스터)

검색·블로그·플레이스와 연결돼 네이버 안에서 영상 노출을 받는 숏폼. 메타/유튜브용 영상을 **그대로 재활용**하므로 추가 제작비 ≈ 0.

**운영 강도:** 주 2~3건 (릴스/쇼츠와 동일 영상 업로드)

**전략:**
- **세로 9:16, 7~15초, 첫 1초 후킹** 필수
- **킬러 콘텐츠 = 비주얼 변신물:**
  - 방구석 드레스 투어 before→after
  - AI 메이크업 시뮬 변신
  - 청첩장 0원→완성 타임랩스
- **정보성 클립:** "결혼 준비 1초에 끝내는 법", "스드메 호구 안 당하는 체크포인트 3"
- 네이버 클립 **챌린지/주제 태그** 적극 활용, 설명란에 블로그 글 링크 연결
- **클립 ↔ 블로그 상호 연결**로 네이버 내 체류·신뢰 강화

## 4. 유튜브 광고 — 구글애드 30만원 (전환 검증 + 인지도)

소액이므로 "광범위 인지"보다 **타겟 좁혀 전환·CPI 검증**에 집중. 진짜 목적 = **어떤 소재/타겟이 설치를 부르는지 데이터 확보** → 예산 증액 시 근거.

**캠페인 설계:**
- **포맷:** 건너뛸 수 있는 인스트림 + **유튜브 쇼츠 노출** (CPV 저렴, 숏폼 소재 재활용)
- **앱 설치 캠페인(UAC):** 목표가 설치면 Google App Campaign이 자동 최적화 → 소액에 적합
- **타겟팅 (좁게):**
  - 지역 한국 / 연령 25~40 / 성별 전체
  - 라이프이벤트 **"약혼/결혼 예정"**(결혼 예정자 직접 타겟 — 핵심)
  - 맞춤 잠재고객: "결혼 준비", "웨딩홀", "스드메", "청첩장" 관심·검색
- **소재 A/B (3종):** ①드레스 투어 임팩트 ②AI 챗봇 "이것까지 다 해줘?" ③청첩장 무료 제작 — **15초 이내, 첫 5초에 핵심 + 브랜드명**
- **예산 배분:** 30만원을 **2~3주 분할**(일 1.5~2만원) → 초반 데이터로 승자 소재에 집중
- **CTA:** "지금 무료로 시작" → 플레이스토어/웹앱 딥링크
- **측정:** CPI(설치당 비용), 가입 전환율, 소재별 CTR → 표로 정리해 증액 판단 근거화

**기대치(현실):** 30만원으로 대량 설치는 어려움. **"설치 1건당 얼마인지" 숫자를 얻는 게 성공.** 인지도는 쇼츠 노출수로 보조 측정.

## 5. 메타 광고 (인스타·페이스북) — 비주얼 전환의 본진

드레스 투어·청첩장 같은 비주얼 자산은 인스타와 궁합 최상. 예산 미정이므로 **2-트랙**.

**트랙 A — 무료(유기적) 먼저 (예산 0, 지금 당장):**
- **인스타 계정 = 브랜드 허브.** 릴스(클립/쇼츠 영상 재활용) + 피드(청첩장 템플릿·예산 인포그래픽 캐러셀)
- **릴스 주 3~4회**, 해시태그: #예비신부 #결혼준비 #스몰웨딩 #셀프웨딩 #청첩장 #스드메
- **저장·공유 유발 콘텐츠**(알고리즘 핵심): "결혼 준비 체크리스트", "지역별 웨딩 평균 예산표" 캐러셀
- 프로필 링크 = 딥링크, 스토리 하이라이트로 기능별 정리

**트랙 B — 유료 (예산 풀리면, 권장 시작액 일 1만원~):**
- **목표:** "앱 설치" 또는 "전환"(가입) 최적화 (Meta SDK/픽셀 이벤트 연동 선행)
- **타겟:**
  - 핵심: 한국 25~40, 관심사 "결혼/약혼/웨딩/신혼"
  - **유사 타겟(Lookalike):** 베타 가입자·앱 이벤트 기반 1~3% (데이터 쌓인 뒤 강력)
  - 리타게팅: 웹/앱 방문 후 미가입자
- **소재:** 릴스 영상 + 캐러셀(before/after, 무료 청첩장) — **인스타 릴스 면 우선**(CPM 효율)
- **A/B:** 드레스투어 vs 청첩장무료 vs AI상담 / 영상 vs 캐러셀
- **배분 원칙:** 구글 30만원과 동일 소재로 돌려 **채널별 CPI 비교** → 효율 좋은 쪽 증액

**선행 작업:** 메타 픽셀/SDK 이벤트(설치·가입·하트구매) 연동 — 없으면 유료 최적화 불가.

- **1주차:** 공통 토대(UTM·딥링크·픽셀) 세팅, 인스타/블로그 개설, 원본 6종 제작, 카페 가입·활동 시작

- **이후:** 효율 채널에 예산 증액(데이터 기반), 유사타겟 광고 확장, 블로그 SEO 자산 누적

## 부록 — 즉시 실행 도구

### A. UTM 네이밍 규칙

```
?utm_source={naver_blog|naver_cafe|naver_clip|youtube|meta}
&utm_medium={organic|cpc}
&utm_campaign={softlaunch_2026q3}
&utm_content={소재명: dresstour|invitation|aichat}
```

### B. 블로그 글 4편 초안 (제목 + 목차)

1. **"2026 결혼 준비 순서 총정리 (예비부부 체크리스트)"** — 시기별 할 일 / 스드메 순서 / 예산 잡기 / "Dewy로 자동 관리" CTA
2. **"스몰웨딩 준비 비용, 진짜 얼마 들까? (실제 견적 공개)"** — 한옥·하우스·카페 비교 / 항목별 비용 / 일반웨딩 대비 절감
3. **"셀프 청첩장 무료로 만드는 법 (모바일·종이 둘 다)"** — 템플릿 고르기 / 문구 AI 추천 / QR·하객관리 / 인쇄용 PDF
4. **"AI 드레스 투어 써봤다 — 내 사진으로 드레스 미리보기 후기"** — before/after / 사용법 / 솔직 후기

### C. 숏폼 스크립트 6종 (7~15초)

1. **드레스투어:** (1초) "드레스샵 가기 전에 이것부터" → before/after 전환 → "Dewy 방구석 드레스 투어"
2. **청첩장:** "청첩장 5만원? 0원에 만들었습니다" → 제작 타임랩스 → 완성본
3. **AI상담:** "결혼 준비 막막할 때" → 챗봇에 질문 → 답변 PDF 생성 장면
4. **예산:** "우리 결혼 예산, 평균보다 많이 쓰는 중?" → 지역 평균 비교 화면
5. **스몰/셀프:** "스몰웨딩인데 일반웨딩 앱 쓰지 마세요" → 페르소나 전환 화면
6. **메이크업:** "내 얼굴에 웨딩 메이크업 미리보기" → 시뮬 변신
