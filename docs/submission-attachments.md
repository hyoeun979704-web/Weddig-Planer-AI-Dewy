## 권장 진행 순서

1. **이번 주 안에** (출시 직후):
   - A1 이력서, A5 외부 검증 증빙, A6 GitHub 캡처
   - C1 화면 캡처 세트(최소 10장), C3 AI 결과물 샘플
   - F1·F2 URL 정리 (이미 게시 완료)

2. **2~4주 안에** (Phase 2 베타):
   - D1 인터뷰 결과 (5~10명)
   - D2 정량 데이터 (GA4/Mixpanel 첫 측정치)
   - E1 매출·비용 추정 엑셀

3. **Phase 3 진입 시**:
   - B1 사업자등록증, B2 통신판매업
   - E3 자금 사용 계획서 (정부사업 양식)

4. **시드 IR 진입 시**:
   - H1 IR 덱 PPT 변환
   - H3 데이터 룸 구축
