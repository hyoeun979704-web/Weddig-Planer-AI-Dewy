# mobile-modern-01 — 모바일 모던 세로

Phase 3-C 모바일 청첩장 무료 등급.

## 무드
- 단정·모던·인스타 스토리 비율 (9:16)
- 상단 정사각 사진 + 하단 텍스트 카드

## 캔버스
- **1080 × 1920** (인스타 스토리 표준)
- 배경 #FAF8F4 (워머 화이트)
- 카카오톡·인스타·문자 어디서나 잘 보임

## Figma 작업 가이드

### 배경 PNG (`mobile-modern-01-bg.png`)
- 상단 0~1080 — 사진 자리 (회색 box 또는 빈 채로)
- 하단 1080~1920 — 워머 화이트 #FAF8F4
- (선택) 사진 영역 하단 1/4 에 살짝 다크 그라데이션 (텍스트 가독성)

### 풀시안 PNG (`mobile-modern-01-thumb.png`)
- 위 배경 + placeholder 사진 + 모든 텍스트
- 그리드 미리보기 용도

## QR 슬롯은 없음
모바일 청첩장 자체가 QR 스캔 대상 → 본인 안에 QR 넣을 필요 없음. 종이 청첩장에서만 QR 슬롯 사용.

## 모바일 청첩장 흐름

```
1. 사용자 wizard 입력 + 사진 1장 첨부
2. "청첩장 만들기" → result 페이지
3. "공유 발행" 클릭 → publish_invitation RPC
4. share_slug 자동 발급
5. /i/<slug> 공개 URL → 카카오톡·인스타·문자로 공유
```

## 디자인 메모
- 모바일은 화면 비율 9:16 → 정사각 사진 + 텍스트 영역 구분 명확
- 사진 위 흰 텍스트는 어두운 그라데이션 PNG 가 가독성 보장
- 캘린더 색상은 결혼식 컨셉 핑크 #E0364B (또는 디자이너 선택)
- 인사말 슬롯이 큰 폰트 (24pt) — 모바일에서 잘 읽힘
