한국 표시광고법 + 인용 룰 — 발행 전 체크
================================================================
※ 법적 가이드는 변경됨 — 최신 공정거래위원회 가이드 확인 필요.
※ 발행 전 운영자가 카테고리 확인 후 표기.

[광고·협찬 표기 의무 (한국)]
공정거래위원회 추천·보증 등에 관한 표시·광고 심사지침 기준:

▶ 표기가 반드시 필요한 경우
- 업체로부터 금전·물품·서비스를 제공받음 (할인·무료 포함)
- 업체와 제휴·파트너십 관계
- partner_deals 테이블의 상품·서비스를 카드화

▶ 표기 위치
- 캡션 본문 첫 줄 또는 마지막에 명확하게 (해시태그 사이 숨김 금지)
- 카드 이미지 본문 안에도 작은 라벨 권장 (우상단 #광고 칩)

▶ 표기 문구 (택1)
- "#광고"        — 가장 기본
- "#협찬"        — 일부 제공
- "#유료광고포함"   — 명확한 광고
- "Dewy 의 제휴 업체입니다" — 본문 한 줄
※ "#소정의 원고료를 받았습니다" 도 가능하나 가독성 ↓

▶ 어떤 콘텐츠에 표기?
- partner_deals 출처:                 항상 #광고 표기
- promotional_events 자체 이벤트:      "Dewy 이벤트" (자사 → 광고 표기 의무 ✗)
- tips_blogs / tips_instagrams 인용:  광고 X / 출처 표기 O (아래)
- 일반 정보·앱 기능 소개:             표기 의무 없음

[저작권 · 인용 (tips 데이터 활용 시)]
tips_blogs · tips_instagrams · tips_videos 는 외부 콘텐츠를 큐레이션한
것이므로, 카드뉴스에 인용 시 명확한 출처 표기 필요.

▶ 인용 가능한 범위
- 정보·팁의 요약·재구성 ✅
- 작성자 이름·블로그명 표기 시 짧은 발췌 ✅
- 이미지·썸네일 직접 사용 ❌ (라이센스 없음)
- 본문 통째 복붙 ❌ (저작권 침해)

▶ 출처 표기 형식 (카드 본문)
"@blogger_name · blog.naver.com" — 카드 우측 하단 작은 텍스트
또는 캡션에 "정보 출처: @blogger_name · 원문은 프로필 링크"

▶ 인스타그램 게시물 인용
- @계정명 멘션 필수
- 가능하면 해당 계정 사전 동의 (DM 한 통이면 OK)
- 원본 이미지는 절대 직접 게재 ✗ — 텍스트 요약만

▶ 유튜브 영상 인용
- 채널명·영상 제목 표기 ✓
- 썸네일 직접 사용 ✗
- "이 정보는 YouTube @channel_name 의 영상을 참고했어요"

[저작권 안전한 콘텐츠 소스]
- 자체 작성 (운영자·AI 카피)
- 앱 데이터·통계 (지역별 평균 등)
- 자체 촬영 이미지·앱 스크린샷
- 라이센스 명시된 무료 이미지 (Unsplash, Pexels — 출처 표기 권장)
- public domain (창작 후 70년 경과)

[Dewy 자체 콘텐츠 표기]
- "출처: Dewy 앱 데이터" — 평균·통계
- "스크린샷: Dewy 앱" — 기능 카드

[광고심의 / 의료법 / 식약처]
다음 주제는 한국 광고법상 추가 심의 대상이므로 카드뉴스 만들 때 주의:
- 의료·시술 (피부·시술 추천 ✗ → 일반 뷰티 가이드만)
- 다이어트 효과 단정 ✗
- 가격 단정 광고 — "최저가 보장" 류 ✗

[발행 전 체크리스트]
□ 광고·협찬 여부 확인 → 필요 시 #광고 표기
□ 외부 정보 인용 시 출처 표기
□ 이미지 라이센스 (자체 / 무료 라이센스 / 사전 동의)
□ 가격·평균은 "지역·개인차" 전제 명시
□ 의료·다이어트 단정 표현 없는지

[Claude Code 활용 메모]
- source_type === 'partner_deal'  → 자동으로 캡션 첫 줄에 "#광고" 삽입
- source_type === 'tip_blog' / 'tip_instagram' / 'tip_video' → 출처 자동 표기
- source_type === 'promotional_event' (자사) → 광고 표기 없이 진행
- 의료·시술·다이어트 키워드 감지 시 자동 거부 + 경고
