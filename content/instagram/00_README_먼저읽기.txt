1) 레포에 이 폴더를 둔다 (예: /content/instagram/)
2) 00_ClaudeCode_지침.md  → Claude Code에 줄 작업 지시(또는 CLAUDE.md에 반영)
3) 01~08 .txt  → Claude Code가 참조하는 지식/콘텐츠 문서
4) 클로드디자인_폼문안.md → Claude Design에서 카드 디자인 시스템(색·폰트·컴포넌트)
   코드 생성할 때 폼에 붙여넣는 입력
5) [운영자 작성] 자리(가격·출시일·지역평균·실제 기능명)를 채운다

사용 예 (Claude Code에 입력):
  "/content/instagram 문서들 참고해서 '결혼 준비 첫 단계' 주제로
   4:5 카드뉴스 만들어줘. 스크린샷은 /assets/shots 에 있어."

※ 모든 파일 UTF-8 한글. 가격·추천·평균은 '지역·개인차 전제' 톤으로 작성.
