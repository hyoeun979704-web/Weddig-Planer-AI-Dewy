================================================================
[09] 시즌·월별 발행 캘린더  (Dewy 공식계정)
축2·축4 — 계절·시즌 콘텐츠 사이클
================================================================
※ 한국 결혼 시즌 패턴 기반. 지역·트렌드 차이 있음.
※ 운영 시작 후 1~2개월 실데이터로 빈도·요일 조정.

[연간 결혼 시즌 흐름]
- 봄(3~5월) · 가을(9~11월) — 본식 성수기
- 여름(6~8월) — 비수기, 신혼여행 / 청첩장 / 다음 봄 예약
- 겨울(12~2월) — 호텔식 · 실내 / 다음 봄 준비 시작

[월별 콘텐츠 큐레이션]

3월: 봄 시즌 인 — 옥외·정원 웨딩홀, 봄 컬러 드레스, 야외 촬영 팁
4월: 본식 피크 시작 — 본식 D-30 체크리스트, 부케·꽃, 5월 신부
5월: 옥외 웨딩의 정점 — 야외 의전, 우천 대비, 신혼여행 항공권 팁
6월: 비수기 진입 — 가을 본식 위한 D-6개월 준비, 여름 신부 메이크업
7월: 신혼여행 시즌 — 휴가철 신혼여행 추천, 더위 대비 본식 팁
8월: 여름 비수기 — 가을 D-3개월 본식 디테일, 청첩장 발송 타이밍
9월: 가을 본식 시작 — 단풍 야외 촬영, 호텔식 vs 컨벤션 비교
10월: 가을 피크 — D-1개월 잔금·동선, 한복 + 가을 컬러
11월: 가을 마지막 — 본식 마무리, 다음 봄 웨딩홀 예약 시작
12월: 호텔식·실내 — 연말 본식, 다음 해 예산 짜기, 신년 결심
1월: 다음 해 본격 준비 — D-9~12개월 첫 단계, 상견례, 양가 합의
2월: 봄 시즌 임박 — 봄 본식 D-3개월 잔금, 야외 우천 대비

[축별 빈도 권장 (월 기준)]
- 축1 (앱 기능·사용법):       주 1회 = 월 4회   (25%)
- 축2 (꿀팁·정보 — 메인 엔진): 주 2회 = 월 8회   (50%)
- 축3 (활용·사례):            주 1회 = 월 4회   (25%)
- 축4 (소식·업데이트·이벤트):  부정기 (시즌·캠페인 시)
→ 운영 초기는 주 2~3회로 시작 → 안정되면 주 4~5회

[발행 시간대 — 확정]
- 매일 오전 10:00 (KST) 1회 발행
- 주말 포함, 시즌·캠페인 무관 동일 시간
- pg_cron 표현: '0 10 * * *' (Asia/Seoul)
- 자동 발굴(4단계)은 매일 09:00 KST 에 draft 생성 → 운영자 09:00~10:00 검토 →
  승인 시 10:00 자동 발행
- 발행 후 인사이트로 추후 보정 (2~3개월 운영 후 시간대 재검토)

[시즌 캠페인 예시]
- 1월: "올해 결혼해요" 신년 결심 시리즈 (5종 캐러셀)
- 5월: "5월 신부" 옥외 본식 가이드 (3편 연재)
- 9월: "가을 본식 D-1개월" 카운트다운 (4편 연재)
- 12월: "내년 결혼 예비부부" D-12개월 첫 단계 시리즈

[연결 메시지]
- 시즌마다 마지막 카드는 "Dewy 체크리스트가 시기별로 알려줘요"

[Claude Code 활용 메모]
- 주제 입력 시 현재 월·시즌을 함께 전달 → 자동 시즌 매칭
- "축2 / 5월 / 옥외 웨딩홀" 형식 태그로 분류
- promotional_events DB 와 연동 시: 이벤트 시작일 D-7 ~ D+3 발행
