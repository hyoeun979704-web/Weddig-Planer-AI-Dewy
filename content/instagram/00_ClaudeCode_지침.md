## ■ 처리 순서
1) 주제 → 축 → 템플릿: 기능소개→기능카드 / 단계·사용법→번호 프로세스 / 꿀팁·체크→리스트 / 예산·평균→비교 카드 / AI시뮬·청첩장→쇼케이스(전후·목업)
2) 슬라이드 설계 (문제→정보/기능→방법→안심→CTA), 한 카드 한 메시지
3) 카피: 톤 준수, 커버 12자 내외, 본문 소제목1줄+설명1~2줄
4) 이미지 배치 ★
   - 스크린샷/목업 있음: 파일명·순서·라벨로 기능 카드 슬롯에 매핑, 모호하면 1회 확인
   - 없음: 타이포/아이콘 카드 또는 "[스크린샷 자리—비움]", 필요시 목업 생성 지시(텍스트)
5) 출력: 분석 / 이미지맵 / 슬라이드별 구성 / 캡션+해시태그 / (템플릿코드 있으면) 카드 HTML
