# 10 · 카드 안전영역 + 디자인 토큰 (Dewy)

> 01·클로드디자인_폼문안 의 색·폰트 위에 **좌표·간격·굵기** 디테일을 더한 구현 사양.
> Satori / Figma / Canva 어느 도구로 만들어도 동일 결과가 나오도록 픽셀 단위 명시.

## ■ 캔버스 · 안전영역

```
캔버스: 1080 × 1350 (4:5 세로)
┌────────────────────────────────┐  0
│  TOP SAFE      ↕ 120px         │
│  ┌──────────────────────────┐  │  120  ← 상단 안전선
│  │                          │  │
│  │   본문 영역              │  │
│  │   1080 - (좌우 100×2)    │  │
│  │   = 880px wide           │  │
│  │                          │  │
│  │                          │  │
│  └──────────────────────────┘  │  1230 ← 하단 안전선
│  BOTTOM SAFE   ↕ 120px         │
│  (인스타 캐러셀 점·하단 UI 자리)│
└────────────────────────────────┘  1350
```

| 영역 | 값 | 비고 |
|---|---|---|
| 좌우 마진 | 100px | 텍스트·이미지 모두 |
| 상단 안전선 | 120px | 핵심 정보는 이 아래 |
| 하단 안전선 | 120px | 워터마크·페이지번호만 허용 |
| 본문 가용폭 | 880px | 줄당 한글 16~22자 권장 |
| 그리드 베이스 | 8px | 모든 spacing 의 배수 |

⚠️ **인스타 캐러셀 점(dot) 표시는 하단 중앙 약 60px 영역을 가림** — 하단 100~120px 에는 핵심 텍스트 금지.

## ■ Spacing 토큰

```
xs   = 4px    (인라인 간격)
sm   = 8px    (라벨 ↔ 값)
md   = 16px   (블록 내부 padding)
lg   = 24px   (제목 ↔ 본문)
xl   = 40px   (섹션 간 분리)
2xl  = 64px   (커버 큰 여백)
```

## ■ Radius 토큰

```
sm   = 12px   (작은 칩·해시태그)
md   = 20px   (정보 카드)
lg   = 28px   (큰 카드·일러스트 박스)
pill = 9999px (라운드 버튼·태그)
```

## ■ Border / Line 토큰

```
hairline = 0.75px  (악센트 골드 라인용 — 너무 두꺼우면 트림 손상)
thin     = 1.5px   (카드 외곽)
default  = 2px     (강조 박스)
focus    = 3px     (CTA 보더)
```

## ■ 타이포 스케일 (Pretendard 기준 / 1080×1350 px)

| 역할 | 크기 | weight | line-height | 사용처 |
|---|---|---|---|---|
| 커버 헤드라인 | **84px** | 700 | 1.15 | 표지 1번 카드 (12자 이내) |
| 커버 서브 | 32px | 400 | 1.4 | 표지 보조 한 줄 |
| 본문 H1 | **56px** | 700 | 1.25 | 본문 카드 큰 제목 |
| 본문 H2 | 40px | 600 | 1.3 | 본문 카드 소제목 |
| 본문 P | 28px | 400 | 1.6 | 설명 (한 카드 2~4줄) |
| 라벨 / 칩 | 22px | 500 | 1.2 | 해시태그·번호 |
| 워터마크 | 24px | 500 | 1 | Cormorant 세리프 |
| 페이지번호 | 22px | 500 | 1 | "01 / 07" |

**모바일 가독성 최소선**: 본문 28px 이하 금지 (인스타 피드 축소 시 22~24sp 로 보임).

## ■ 색 대비 검증 (WCAG AA)

| 조합 | 비율 | 사용 가능 |
|---|---|---|
| #3E3438 ↔ #FFF9F6 | 13.2:1 | ✅ 본문 |
| #3E3438 ↔ #FDF1F3 | 12.5:1 | ✅ 본문 |
| #8C8085 ↔ #FFF9F6 | 4.6:1 | ✅ 보조 (AA 통과) |
| #F4A7B9 ↔ #FFF9F6 | 1.9:1 | ❌ 텍스트 금지 / 면적·아이콘만 |
| #E0859A ↔ #FFF9F6 | 3.0:1 | ⚠️ 큰 텍스트(28px+ bold) 만 |
| #C9A86A ↔ #FFF9F6 | 2.5:1 | ❌ 텍스트 금지 / 라인·아이콘만 |

**규칙**: 본문 색은 항상 `#3E3438` 또는 `#8C8085`. 핑크·골드는 강조 배경·라인·아이콘 전용.

## ■ 카드별 푸터 (모든 카드 공통)

```
좌측 하단 (100, 1230):  Dewy 워드마크 (Cormorant, 24px, #3E3438)
우측 하단 (980, 1230):  페이지번호 "01 / 07" (Pretendard 22px, #8C8085)
```

## ■ 로고 사용 룰

- 워드마크 "Dewy" 만 사용 — 단독 하트 아이콘은 표지 1번 카드에서만
- 표지 1번: 상단 중앙 (540, 200) 에 워드마크 48px
- 본문 카드: 좌측 하단 워드마크 24px (위 푸터 참조)
- 색상: `#3E3438` 검정톤이 기본 / 핑크 배경 위에서는 `#FFFFFF`

## ■ 아이콘 사용 룰

- **라인형 단색 only** (Lucide / Phosphor light)
- 크기: 본문 48px, 워드마크 옆 28px
- 색상: `#3E3438` 기본 / 강조 시 `#F4A7B9`
- 면(solid) 아이콘·이모지 금지 (광택·강한 채도 → 톤 깨짐)

## ■ 이미지 슬롯 (스크린샷 / 시뮬 결과)

```
정사각형 슬롯: 480×480, radius lg (28px)
모서리: 안쪽에 1.5px 두께 #00000010 보더
배경: 슬롯 안쪽은 #FFFFFF, 밖은 카드 배경
캡션: 이미지 아래 16px 간격 + 28px Pretendard 400
```

## ■ Claude Code 활용 메모

- 모든 좌표·크기는 **1080×1350 기준** — 다른 해상도 만들 때 비율 변환
- Satori 사용 시 `width: 1080, height: 1350, fonts: [Pretendard, Cormorant]`
- 본문 28px 미만 출력 금지 (자동 검사 권장)
- 핑크/골드를 텍스트 색으로 쓰는 시도가 나오면 거부
