================================================================
[02] 앱 기능 · 사용법  (Dewy)
축1(기능·사용법) 카드 작성 시 참고
================================================================
※ 실제 화면·버튼명·세부 동작은 앱 기준으로. 모르면 [운영자 작성].

[기능별 한 줄 + 카드화 포인트]
1) 체크리스트·D-Day
   - 결혼까지 할 일을 시기별로 자동 정리, 디데이로 일정 관리
   - 카드: "뭐부터?" 막막함 → 체크리스트가 순서대로 알려줘요

2) 예산·양가 분담
   - 항목별 예산, 신랑/신부·양가 분담, 지역별 평균과 비교
   - 카드: 평균 비교 그래프 + "우리는 어디에 얼마?"

3) 맞춤 추천 (웨딩홀·스드메·예복·한복·예물·신혼여행)
   - 조건 입력 → 맞춤 추천
   - 카드: "조건만 넣으면 추천이 와요" / 카테고리별 시리즈

4) AI 드레스·메이크업 시뮬레이션
   - 사진으로 미리 입어보고 메이크업 미리보기
   - 카드: Before/After 쇼케이스 (시뮬 결과 이미지)

5) 모바일 청첩장
   - 앱에서 바로 제작·공유
   - 카드: 청첩장 목업 쇼케이스 + 제작 단계

[사용법 카드 공식 — 번호 프로세스]
01 가입/정보입력 → 02 기능 실행 → 03 결과 확인/저장
→ 실제 단계·화면은 [운영자 작성/스크린샷]

[Claude Code 활용 메모]
- "OO 기능 소개" → 기능 카드(아이콘/스크린샷+제목+설명)
- "OO 사용법" → 번호 프로세스 + 스크린샷 슬롯
- "AI 시뮬 / 청첩장" → 쇼케이스 템플릿(전후·목업)
- 화면 캡처 있으면 해당 슬롯에 배치, 없으면 목업 placeholder
