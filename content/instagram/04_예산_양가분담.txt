================================================================
[04] 예산 · 양가 분담  (Dewy)
축2(꿀팁·정보) — 비교/평균 카드 소재
================================================================
※ 금액·평균은 지역·시즌·개인차 큼.

[평균 비용 출처 — 카드 작성 시 반드시 표기]
- 1차 출처: 한국소비자원 결혼비용 조사 (2026.02 발표)
- 2차 출처: Dewy 앱 내 사용자 입력 데이터 (지역별 집계, 익명)
- 카드 푸터에 작은 글씨로: "출처: 한국소비자원 2026.02 · 지역·개인차 있음"
- 한국소비자원 데이터에 없는 항목은 [대괄호] 로 비우고 운영자 확인

※ Dewy 앱 내 평균 비교 화면에서도 같은 출처를 표기하므로,
   카드 데이터와 앱 데이터가 일치해야 함 — 앱 캡처가 더 정확.

[예산 큰 항목]
- 예식: 웨딩홀(대관·식대·꽃), 스드메, 예복·한복
- 혼수·신혼집, 예물·예단, 신혼여행, 모바일/종이 청첩장 등
→ "어디에 얼마"가 막막 → Dewy 항목별 예산·평균 비교로 해결

[양가 분담]
- 분담은 가정마다 달라 '정답 없음' → 사전 합의가 핵심
- 갈등 포인트: 예단·예물 범위, 신혼집 분담, 식대 부담
→ 카드 톤: 비교·강요 금지. "합의를 돕는 도구" 포지셔닝

[비교/평균 카드 소재 예]
- "결혼 평균 비용, 어디에 가장 많이 쓸까?" (항목 비중)
- "지역별 웨딩홀 평균" ([앱 데이터 인용])
- "예산 짤 때 흔한 실수 3가지"

[주의]
- 특정 금액 단정·과장 금지, "평균/추정·개인차" 표기
- 지역별 평균은 앱 데이터 출처를 전제로

[Claude Code 활용 메모]
- "평균/비용/비중" → 비교·평균 카드(바/도넛)
- "분담/합의" → 리스트 + 다정한 톤
- 수치는 자료에 있는 것만, 없으면 [대괄호]
